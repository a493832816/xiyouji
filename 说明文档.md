# 西游地图交互还原说明

## 1. 交付清单

- `index1.html`：主页面（底图、交互图层、控件、图例、详情面板）
- `styles/base.css`、`styles/map.css`、`styles/poi.css`：样式资源
- `scripts/poi-data.js`：地图数据（地点、路线、地形、图例、区域标签）
- `scripts/map-renderer.js`：渲染模块
- `scripts/map-controls.js`：缩放/拖拽/复位控制模块
- `scripts/map-utils.js`：工具函数模块
- `scripts/main.js`：入口与交互绑定
- `tests/run-tests.html`：测试入口
- `tests/unit.test.js`：单元测试用例
- `tests/integration.test.js`：集成测试用例

## 2. 运行步骤

1. 在当前目录启动任意静态服务器（推荐）：
   - Python：`python -m http.server 8080`
   - Node：`npx serve .`
2. 浏览器访问：
   - 主页面：`http://localhost:8080/index1.html`
   - 测试页：`http://localhost:8080/tests/run-tests.html`

## 3. 依赖列表

- 运行依赖：现代浏览器（Chrome / Firefox / Safari / Edge 最新版本）
- 构建依赖：无（纯 HTML + CSS + JavaScript ES Modules）
- 第三方库：无

## 4. 关键交互说明

- **缩放**：支持按钮缩放与鼠标滚轮缩放
- **拖拽**：按住地图拖动（触屏可平移）
- **复位**：一键恢复到 100% 视图
- **热点悬浮**：显示地点名与区域
- **热点点击**：高亮地点、联动右侧详情并居中定位
- **矢量图层**：路线与地形均由 SVG 绘制，放大不失真

## 5. 已知问题

- 移动端 Safari 在低端机上连续高频缩放可能出现轻微掉帧
- 部分古风字体受系统字体差异影响，字形存在微小视觉差异
- 点击热点跳转目前采用页面内 hash 与详情联动，未接入外部知识库

## 6. 后续扩展接口

- `scripts/poi-data.js`
  - `points`：新增/修改热点
  - `routePath`：扩展主线或分支路线
  - `terrainPaths`：扩展山脉/水系矢量轮廓
- `scripts/map-renderer.js`
  - 可新增专题图层渲染（阵营战争线、事件热度）
- `scripts/main.js`
  - 可扩展热点点击行为（跳转百科、弹窗、音视频）
- `scripts/map-controls.js`
  - 可扩展双指缩放、最小地图鹰眼等高级交互
