# 图例及符号说明

## 1. 图例规范

- **橙色矩形节点（地点）**
  - 含义：山岭、河流、寺观、洞府、郡县等事件发生地
  - 样式：`#f4b463` 填充，深灰描边

- **红色矩形节点（边界/阶段）**
  - 含义：国界、洲界、行程阶段分割点（如“两界山”“天竺国界”）
  - 样式：`#d86a6a` 填充，白色文字

- **深灰箭线（路线方向）**
  - 含义：取经队伍行进方向
  - 样式：`#404a55` 线条，箭头终点标记

## 2. 配色说明

- 主线颜色：`#404a55`
- 地点节点：`#f4b463`
- 边界节点：`#d86a6a`
- 背景底色：`#f8fafc`
- 文字主色：`#1d2830`

以上颜色采用统一规范，便于后续 AI / SVG 二次编辑时保持一致性。

## 3. 符号使用约定

- 单节点代表一个关键站点，不折叠多个地点
- 箭头仅表示流程方向，不代表精确方位角
- 上下折返连接表示跨行延续，保持与原图流程一致
- 节点坐标以 SVG `viewBox` 坐标为准，详见 `节点坐标与名称对照表.csv`
